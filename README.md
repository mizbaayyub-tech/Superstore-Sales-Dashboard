# Superstore-Sales-Dashboard
Interactive Sales Dashboard created in Power BI using the Superstore dataset.
## Project Overview
This project presents an interactive Sales Dashboard created in Power BI using the Superstore Sales dataset. The dashboard helps business stakeholders analyze sales performance, customer information, product distribution, and regional sales trends.

## Tools Used
- Power BI
- Microsoft Excel
- GitHub

## Dataset
Superstore Sales Dataset

## Dashboard Features
- Total Sales Card
- Total Orders Card
- Total Customers Card
- Total Products Card
- Sales Trend Over Time
- Sales by Category
- Sales by Region
- Interactive Slicers (Region, Category, Segment)

## Key Insights
- The West region recorded the highest sales.
- Technology generated the highest sales among all categories.
- Sales increased significantly over the years.
- The dashboard allows users to filter data by Region, Category, and Segment for better analysis.
